❗ Task AWS Developer Track ❗



Perform ETL Operation on AWS

  • 1. Create a bucket(folder) in s3 and upload data in .csv format. Here, upload only small size of data just for testing.
  • 2. After loading the data into s3, navigate towards glue.
  • 3. Add crawler in glue. Adding crawler to glue will help you to take the data from s3 and create metadata in the database table in glue. This metadata will help you to write SQL queries in Athena.
  • 4. Then you can navigate towards athena to write SQL queries to take some insights.
  • 5. While writing the SQL queries you have to save the outputs of sql queries to s3. You can create separate subfolders in the bucket for storing the outputs.
  • 6. Now add a job in glue.
  • 7. Glue Jobs are meant to use spark using python or scala. you can use any python/scala.
  • 8. Use the same s3 bucket to store the output of spark jobs but in different subfolders.
  • 9. Use glue triggers to schedule your glue jobs. These triggers should run on our respective scheduled time.
  • 10. At the end create workflows of the whole setup. Glue provides us a separate section to create workflows.
